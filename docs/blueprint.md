# **App Name**: Aravalli Configurator

## Core Features:

- Furniture Selection: Allow users to select from a range of PVC furniture items, categorized for easy browsing.
- Material Customization: Enable users to select PVC material, color, and texture for furniture items.
- Dimension Adjustment: Provide tools to adjust the dimensions (height, width, depth) of selected furniture pieces.
- AI-Powered Recommendations: Use AI to provide material, and size recommendations to users based on past project completions of a similar nature. LLM acts as a tool, selecting the most relevant considerations for each project
- 3D Preview: Generate a 3D preview of the customized furniture configuration.
- AR View: Augmented Reality feature to visualize the furniture in the user's space via their device's camera.
- Configuration Saving & Sharing: Allow users to save their furniture configurations and share them.

## Style Guidelines:

- Primary color: Deep brown (#4A300E) to reflect the earthy tones of PVC furniture.
- Background color: Off-white (#F5F3F0), providing a neutral backdrop.
- Accent color: Soft orange (#D57B00), analogous to the primary color and suggestive of industry and production.
- Headline font: 'Belleza', a serif font. Body font: 'Alegreya', a serif font.
- Use clear, modern icons to represent different furniture types, materials, and customization options.
- Employ a clean, grid-based layout to showcase furniture options and customization tools.
- Incorporate smooth transitions and subtle animations for an engaging user experience.
